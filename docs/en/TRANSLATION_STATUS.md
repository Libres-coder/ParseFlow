### How to Contribute Translations

1. **Choose a pending document**
2. **Follow translation principles**:
   - Preserve all technical terms
   - Keep code examples as-is
   - Maintain Markdown formatting
   - Update internal links

3. **Translation principles**:
   - Accuracy over literal translation
   - Keep technical terms in English when appropriate
   - Preserve code comments in original language
   - Maintain consistent terminology

4. **Submit Pull Request**

- [ ] All content translated
- [ ] Technical terms consistent
- [ ] Code examples unchanged
- [ ] Links updated
- [ ] Formatting preserved
- [ ] Language switcher added
